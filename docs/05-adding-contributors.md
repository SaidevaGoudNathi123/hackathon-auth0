# How to Add a Friend as a Contributor (Collaborator) to This Repository

This guide explains how to invite someone to collaborate on this GitHub repository so they can push code, create branches, and contribute directly.

---

## Prerequisites

- You must be the **repository owner** or have **Admin** access to the repository.
- You need your friend's **GitHub username** or the **email address** linked to their GitHub account.

---

## Steps

### 1. Go to the Repository Settings

1. Open your browser and navigate to the repository:
   `https://github.com/SaidevaGoudNathi123/hackathon-auth0`
2. Click the **Settings** tab (top-right area of the repository page).

### 2. Open the Collaborators Section

1. In the left sidebar, click **Collaborators** (under the *Access* section).
2. GitHub may ask you to confirm your password or complete two-factor authentication.

### 3. Invite Your Friend

1. Click the **Add people** button.
2. In the search box, type your friend's **GitHub username** or **email address**.
3. Select their account from the dropdown list.
4. Choose a **permission level**:
   - **Read** – Can view and clone the repo.
   - **Triage** – Can manage issues and pull requests (no code push).
   - **Write** – Can push code and manage branches. *(Recommended for contributors.)*
   - **Maintain** – Can manage the repo without destructive actions.
   - **Admin** – Full control, including settings and deleting the repo.
5. Click **Add [username] to this repository**.

### 4. Your Friend Accepts the Invitation

- Your friend will receive an **email invitation** from GitHub.
- They can also accept it directly on GitHub by visiting:
  `https://github.com/SaidevaGoudNathi123/hackathon-auth0/invitations`
- Once accepted, they will have access based on the permission level you selected.

---

## Managing Collaborators

- To **change a collaborator's permissions**, return to **Settings → Collaborators**, find their name, and update the role.
- To **remove a collaborator**, click **Remove** next to their name in the same page.
